# Yrobot 学习graphql的demo  
## 项目结构  
目前主要分为两个项目：server和client     
server：graphql的服务   
client：使用apollo+React搭建的SPA，来请求graphql-server，展示结果数据  
## 运行demo
__运行server：__  
```
1. 当前目录新建terminal
2. 运行 sh ./startServer.sh
```
__运行client：__  
```
1. 当前目录新建terminal
2. 运行 sh ./startClient.sh
```
